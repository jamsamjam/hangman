## Hangul Hangman

A Hangman Game where you can learn essential basic Korean words.

<br>

💡 Remarks

1. 겹자음 is treated separately. For example, for ㄶ, either ㄴ or ㅎ is considered as a correct guess. (Note that it is not the case for 쌍자음 like ㅆ.)
